How to run
==========

This package provides users with the capabilities of running and simulating PPFL on either a single machine, a cluster, or heterogeneous machines.

Serial run
----------

.. code-block:: console

    $ python ./my_appfl.py


MPI
---

.. code-block:: console

    $ mpiexec -np 5 python ./my_appfl.py
