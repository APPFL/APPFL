On Amazon Web Services Platform
========================

This describes how to set up ``appfl`` `gRPC <https://grpc.io>`_ server to run on `Amazon Web Services <https://aws.amazon.com/>`_ platform, while the clients are training the model locally. In this tutorial, we describe the steps to run a simple example by using the following services:

.. note::
