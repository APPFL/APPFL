# ARVO course: Eye Disease classification with AI-READI

This directory contains Jupyter notebooks for the ARVO course on AI in Eye Disease classification using data from the AI-READI dataset. The notebooks cover various topics including data preprocessing, classification, and federated learning.

**Classification Notebook:**

- `1-ARVO_Classification.ipynb`

**Federated Learning Notebooks:**

- `APPFL_Server_AI_READI.ipynb` -> Server notebook for federated learning training

- `APPFL_Client1_AIREADI.ipynb`, `APPFL_Client2_AIREADI.ipynb` -> Clients notebook for federated learning training

This `slides` subdirectory also contains three slides in PDF format used in the ARVO course presentation:

- `slides/1-intro-to-dl.pdf`: Introduction to Deep Learning
- `slides/2-handson-classification.pdf`: Hands-on Classification with AI-READI
- `slides/3-handson-fl.pdf`: Hands-on Federated Learning with AI-READI
